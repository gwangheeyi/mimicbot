1. Gazebo 월드에 camera_test 존재
2. gz topic -l에서 영상 토픽 확인
3. ros_gz_bridge 실행
4. ros2 topic list에서 카메라 토픽 확인
5. web_video_server 실행
6. Flutter WebView에서 확인



1. omx-ai gazebo 실행

cd ~/mimicbot/ros2_ws

source /opt/ros/jazzy/setup.bash
source install/setup.bash

ros2 launch open_manipulator_bringup open_manipulator_x_gazebo.launch.py


2. Gazebo 영상을 ROS2로 연결

cd ~/mimicbot/ros2_ws
source /opt/ros/jazzy/setup.bash
source install/setup.bash

ros2 run ros_gz_bridge parameter_bridge \
  /omx_front_camera/image@sensor_msgs/msg/Image@gz.msgs.Image \
  /omx_front_camera/camera_info@sensor_msgs/msg/CameraInfo@gz.msgs.CameraInfo


3. ROS2에서 카메라 토픽 확인

source /opt/ros/jazzy/setup.bash
source ~/mimicbot/ros2_ws/install/setup.bash

ros2 topic list -t | grep omx_front_camera

4. web_video_server 실행

source /opt/ros/jazzy/setup.bash
source ~/mimicbot/ros2_ws/install/setup.bash

ros2 run web_video_server web_video_server


설치되지 않았다면:

sudo apt update
sudo apt install ros-jazzy-web-video-server

5. Windows 브라우저에서 영상 확인

http://localhost:8080/stream?topic=/omx_front_camera/image

전체 토픽 선택 화면은:

http://localhost:8080
